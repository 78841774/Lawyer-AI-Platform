# v7.30 Codex Training Scheme & Artifact Loader

This file is an alias document for the v7.30 training artifact loader. The full implementation note is in `docs/Codex-Training-Scheme-Multi-Level-Case-Cause-Artifact-Loader-v7.30.md`.

v7.30 defines Codex training as metadata artifact generation and loading, not model fine-tuning. It adds multi-level case-cause taxonomy metadata, experience package manifests, Skill manifests, evaluation / gate / test case manifests, loading manifest metadata, case-cause fallback matching, Skill Context dry-run, frontend loader UI, and regression checks.

Safety boundary: no real case material, no raw OCR, no provider call, no API key read, no open-case training, no Skill auto-publish, no final legal opinion, no final report, no real PDF/DOCX, no public link, no email, and no external delivery.
