# AIHome.law Personal Production One-Pager v7.9

## 产品定位

AIHome.law Personal Production Pilot 是面向律师个人生产验证的受控工作流展示骨架。它将 AI 草稿、材料处理、mock 信息核验、Experience Package、受控案件生产、交付包草案、Source Trace、Lawyer Review 和 Final Lock 串联为可演示流程。

## 适用场景

- 律师个人工作流验证。
- 内部产品演示。
- 受控 AI 辅助草稿流程说明。
- 来源追踪与律师复核门禁展示。
- 非真实案件的培训和试点演示。

## 核心能力

- 个人生产总控台：展示 readiness、runtime registry 和安全边界。
- Showcase Pack：展示 Story Flow 与 Pilot Sample。
- Delivery Packet：展示交付包草案、来源追踪和 Final Lock。
- Case Production：展示受控案件生产流程与 review gate。
- Skill Studio：展示经验包和技能候选草案。
- Intelligence Gateway：展示 mock 法律与企业信息核验。
- Material Runtime：展示 metadata-only 材料解析和 OCR preview。
- AI Gateway：展示 draft-only prompt preview 和 mock run。

## 安全边界

- mock-first。
- provider-gated。
- metadata-only。
- draft-only。
- 律师复核必需。
- 来源可追踪。
- final-lock-required。
- 不生成最终法律意见。
- 不生成最终报告。
- 不自动对外交付。
- 不发送邮件。
- 不生成真实 PDF/DOCX。

## 当前阶段

当前为 Personal Production Alpha / Pilot 展示阶段，适合说明产品架构、流程组织和安全边界；不作为外部客户交付版本。
