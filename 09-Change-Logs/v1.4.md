# v1.4 Demo Flow Polish

## Scope

* Added `/cases/new` for browser-based case creation.
* Reworked `/cases/[caseId]` into an interactive case detail page.
* Added browser actions for material upload, fact extraction, legal analysis, and report generation.
* Extended frontend API helpers for the full demo flow.
* Added backend CORS support for local frontend ports.
* Added `GET /cases/{case_id}/reports` for case-level report refresh.

## Demo Flow

Users can now complete the MVP flow without curl:

```text
Create Case -> Upload Material -> Extract Facts -> Run Legal Analysis -> Generate Report -> View Report
```

## Verification

```bash
cd Lawyer-AI-Platform-App/frontend
npm install
npm run build
npm run dev -- -p 3001
```

With the backend running on port 8001, open:

```text
http://localhost:3001
```
