# v1.5 Demo Stabilization

## Scope

* Removed the old standalone `07-Case-Service/` directory because the main backend now owns Case Service.
* Added local startup scripts for backend and frontend.
* Added a port cleanup script for ports `8001` and `3001`.
* Added `docs/Demo-Runbook.md` for repeatable local demos.
* Updated the app README with local demo startup and troubleshooting notes.

## Scripts

```bash
bash Lawyer-AI-Platform-App/scripts/kill-ports.sh
bash Lawyer-AI-Platform-App/scripts/start-backend.sh
bash Lawyer-AI-Platform-App/scripts/start-frontend.sh
```
