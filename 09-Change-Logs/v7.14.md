# v7.14 Legal / Enterprise API Live Gateway

## Summary

v7.14 adds the controlled Legal / Enterprise API Live Gateway foundation inside the v7.10-v7.17 Personal Live Intelligence & Controlled Case Analysis large stage. Legal and enterprise live modes remain disabled by default. Dry-run is available, and gated live runs require explicit confirmations. No real legal search or enterprise information adapter is executed in this stage.

## Added

- Legal / Enterprise live provider config metadata.
- Global legal and enterprise live mode gates with default disabled posture.
- Live gate confirmation guard.
- Legal search dry-run metadata flow.
- Enterprise query dry-run metadata flow.
- Gated live run metadata with blocked / adapter unavailable status.
- Query raw content, AI prompt, and citation boundary.
- Metadata-only response sanitizer.
- Live review queue.
- Live source trace metadata.
- Live audit metadata runtime storage.
- `/personal-intelligence/live/*` APIs.
- Frontend Legal / Enterprise API Live Gateway controlled access panel.
- v7.14 lightweight API regression.
- v7.14 documentation.

## Safety

- Legal live mode disabled by default.
- Enterprise live mode disabled by default.
- Provider-gated.
- API key only checked by backend environment lookup.
- API key never returned to frontend.
- Dry-run-first.
- Live call requires explicit confirmation.
- Legal raw results are not returned.
- Enterprise raw results are not returned.
- Citation candidates are metadata only and are not final citations.
- Legal / enterprise results are not injected into AI prompts.
- Fact extraction is not triggered.
- Legal analysis is not triggered.
- Lawyer review required.
- Source trace required.
- No final legal opinion.
- No final report.
- No external delivery.
