# Change Log v0.4

## MVP Architecture

新增 07-MVP-Architecture 目录，将 v0.1 Documentation Foundation、v0.2 System Design 和 v0.3 Implementation Blueprint 进一步收敛为第一阶段 MVP Architecture。

新增文档：

* README.md：说明 v0.4 MVP Architecture 的目标、文档结构和 MVP 原则。
* MVP-Scope.md：定义 MVP 第一阶段功能边界，包括案件管理、材料上传、Fact Extraction、Legal Analysis 和 Report Generation。
* Tech-Stack.md：确定 Frontend、Backend、Database、Storage、AI Runtime 和 Deployment 技术栈。
* Service-Architecture.md：定义 Case Service、Material Service、Fact Service、Legal Service、Report Service 和 Experience Package Service 的拆分方式。
* Database-Schema.md：输出 MVP 核心数据表设计。
* API-Specification.md：输出 MVP REST API 草案。
* Agent-Orchestration.md：设计 Agent 调度架构、任务链路和错误处理策略。
* Development-Roadmap.md：拆分 Phase 1 MVP、Phase 2 Internal Beta 和 Phase 3 Production。
