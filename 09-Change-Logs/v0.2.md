# Change Log v0.2

## System Design

新增 05-System-Design 系统设计目录，将 v0.1 的业务架构文档转化为可研发的系统接口、数据结构和运行时设计。

新增五类核心设计文档：

* Fact Schema：案件事实数据结构设计。
* Legal Reasoning Schema：法律推理数据结构设计。
* Experience Package Format：经验包格式和加载约定设计。
* Runtime Architecture：Workspace Runtime 架构设计。
* API Design：模块 API 设计原则和 v1 接口草案。
