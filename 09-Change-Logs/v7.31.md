## Safety

- Codex training is metadata artifact generation, not model fine-tuning.
- v7.31 uses `source_case_mode=synthetic_closed_case` when no real redacted closed-case sample set is read.
- v7.31 does not read real case materials, raw OCR, API keys, secret values, local sensitive paths, or real identity data.
- v7.31 does not use open or unresolved cases for training.
- v7.31 does not write training sets, update Skills, or publish Skills.
- v7.31 does not generate final legal opinions, final reports, real files, public links, email, or external delivery.
