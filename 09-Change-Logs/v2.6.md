# v2.6 Skill Runtime

## v2.6-A Skill Package Loader

v2.6-A added the backend Skill Package Loader for built Experience Packages.

## v2.6-B Skill-Aware Runtime Integration

v2.6-B connects applied Workspace Skills to backend runtime services.

## Added

* Fact Extraction now detects an applied published skill and loads its Experience Package runtime context.
* Skill-backed facts use the package domain in `fact_type`, such as `contract_dispute_fact`.
* Legal Analysis now adds an `Experience Package` rule reference when a skill is applied.
* Report Generation now records `skill_id` and `package_id` in `reports.source_refs`.
* Generated reports now include Skill Used, Skill ID, and Package ID metadata.

## Runtime Fallback

Cases without an applied skill continue to use the original MVP behavior.

Cases with an applied skill require both the skill and package to be published.
