# v2.5 Workspace Skill Loader

## Scope

v2.5 adds Workspace Skill Loader to connect published Experience Packages to case workspaces.

The platform can now expose published skills to Workspace and record that a case has applied a skill.

## Added

* Added `app/api/workspace_skills.py`.
* Added `app/services/workspace_skill_service.py`.
* Added `app/repositories/workspace_skill_repository.py`.
* Added `case_skill_bindings` table.
* Added `GET /workspace/skills`.
* Added `GET /workspace/skills/{skill_id}`.
* Added `POST /cases/{case_id}/skills/{skill_id}/apply`.
* Added `GET /cases/{case_id}/skills`.
* Added frontend `/skills` page.
* Added Available Skills and Applied Skills sections to the case detail page.

## Loading Rule

Workspace only loads skills when:

```text
skill.status = published
package.status = published
```

## Notes

v2.5 does not run real AI inference.

Applying a skill records a case workspace binding and prepares the case for future Skill-powered runtime behavior.
