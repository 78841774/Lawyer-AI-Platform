# v7.13 OCR / Document Provider Live Gateway

## Summary

v7.13 adds the controlled OCR / Document Provider Live Gateway foundation inside the v7.10-v7.17 Personal Live Intelligence & Controlled Case Analysis large stage. OCR and document live modes remain disabled by default. Dry-run is available, and gated live runs require explicit confirmations. No real OCR or document parser adapter is executed in this stage.

- OCR live mode disabled by default.
- Document live mode disabled by default.
- Provider-gated.

- Raw document content is not returned.

- Fact extraction is not triggered.
