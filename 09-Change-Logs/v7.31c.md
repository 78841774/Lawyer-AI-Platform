# v7.31c Codex Skill Experience Pool & Draft Builder

## 中文定位

复核通过的训练经验进入 Codex Skill 经验池，并生成待人工确认的 Codex Skill 草案。

## Added

- Added Skill Experience Pool import for v7.31b `approved_for_skill_experience` candidates.
- Added pool entry list/detail metadata.
- Added Skill Experience binding metadata.
- Added Codex Skill draft build, list/detail, review, and audit metadata.
- Added v7.31c APIs under `/personal-skill-studio/training-artifacts`.
- Added frontend v7.31c panels to `/personal-skill-studio/training-artifacts`.
- Added Skill Studio, Personal Production, and Personal Production Pilot summaries.
- Added `scripts/regression/check_personal_codex_skill_experience_pool_v731c_apis.sh`.
- Connected the v7.31c regression script to `scripts/regression/run_personal_alpha_regression.sh`.

## Safety

- v7.31c imports only redacted, source-traced, manually approved experience metadata.
- Pending, rejected, changes-requested, non-redacted, or missing-source-trace candidates are rejected from import.
- Codex Skill drafts are `not_publishable`, `not_training_ready`, and require manual confirmation.
- v7.31c does not call providers, read API keys, read or return source content, write formal training sets, trigger real Codex training, publish Skills, generate final legal opinions, generate formal reports, create public links, send email, or trigger external delivery.
