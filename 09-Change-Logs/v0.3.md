# Change Log v0.3

## Implementation Blueprint

新增 06-Implementation-Blueprint 实施蓝图目录，将 v0.1 架构和 v0.2 系统设计转化为研发落地方案。

新增模块说明：

* Frontend Blueprint：Workspace 前端页面结构，包括案件列表、案件详情、材料上传、事实提炼、法律分析、门控评价、报告输出和经验包选择。
* Backend Blueprint：后端服务模块，包括 Case Service、Material Service、Fact Service、Legal Service、Gate Service、Report Service 和 Experience Package Service。
* Database Blueprint：核心数据库表设计，包括 cases、materials、facts、legal_analyses、gate_results、reports、experience_packages 和 audit_logs。
* File Storage Blueprint：文件存储方案，包括原始材料、OCR 结果、分析结果、报告文件、版本管理和路径规范。
* AI Runtime Blueprint：AI 调用运行时设计，包括模型调用、Prompt 装配、Experience Package 加载、缓存策略、上下文构造、日志记录和错误重试。
* Deployment Blueprint：本地开发、测试环境和生产环境部署方案，包括环境变量、Docker、数据库、对象存储、日志和备份。
