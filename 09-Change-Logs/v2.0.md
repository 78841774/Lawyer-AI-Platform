# v2.0 Skill Training Specification

## Scope

v2.0 adds the Skill Training specification for Lawyer AI Platform.

This version defines Skill Factory as the architecture layer that turns real case experience into reusable, evaluated, and publishable Experience Packages.

## Added

* Added `08-Skill-Training/` as the Skill Training module.
* Added Skill Schema for `skill_id`, `skill_name`, `domain`, `jurisdiction`, `version`, `prompts`, `templates`, `evaluation_metrics`, and `metadata`.
* Added Skill Factory training pipeline from Case to Experience Package.
* Added Evaluation Framework with accuracy, consistency, completeness, legal_relevance, and report_quality scoring.
* Added Package Build System defining Experience Package output files.
* Added Skill Versioning rules for `v1`, `v1.1`, and `v2`.
* Added Skill Lifecycle from Draft to Deprecated.
* Added Contract Dispute Skill example.

## Architecture Impact

Skill Training connects:

```text
Training System
  |
  v
Skill Factory
  |
  v
Experience Package
  |
  v
Workspace System
```

Training System remains responsible for capability production.

Workspace System remains responsible for capability consumption.

Skill Factory defines how legal practice experience is extracted, structured, evaluated, versioned, and packaged.

## Output Format

The v2.0 Experience Package output format is:

```text
experience-package/
├── skill.json
├── fact_prompt.txt
├── analysis_prompt.txt
├── report_prompt.txt
├── templates/
└── tests/
```

## Verification

This release contains architecture documentation only.

No runtime code was added.
