# v4.3 Change Log

## v4.3 Controlled Local Material Read Implementation

Implemented controlled local read preview for small text materials.

### Added

- Added controlled local read preview.
- Added local file path guard.
- Added file extension guard.
- Added file size guard.
- Added path outside repo guard.
- Added runtime storage guard.
- Added redacted preview builder.
- Added runtime storage for redacted preview.
- Added API `POST /controlled-material/local-read-preview`.
- Added API `GET /controlled-material/read-preview/{preview_id}`.
- Enhanced `/controlled-material` page.
- Enhanced `/runtime` Controlled Material status.
- Added `docs/Controlled-Local-Material-Read-Implementation-v4.3.md`.

### Safety

- Local-only.
- No production deployment.
- Only `.txt`, `.md`, and `.json` supported.
- No PDF, Word, or image reading.
- No OCR call.
- No LLM call.
- No Legal Search call.
- No DeepSeek live call.
- No raw text committed.
- No real material committed.
- Redacted preview only.
- Redacted preview is stored only in ignored runtime storage.
- Manual lawyer review required.
- A1-A13 remain unchanged.
- A10 remains `争议焦点法律深化分析`.
