### Data-First Decision

Training should not start from raw legacy folders.

Before training or Skill Registry import, legacy data must be reshaped into reviewed package structures:

* Dataset Packages.
* Runtime Rules.
* Prompt Templates.
* Report Templates.
* Evaluation Rubrics.
* Reference-only assets.
