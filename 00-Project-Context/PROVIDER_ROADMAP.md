# Provider Roadmap

## Provider Categories

- AI: OpenAI, DeepSeek, Local Model.
- OCR: PaddleOCR / Baidu AI Studio.
- File Parsing: MinerU, Docling.
- Legal Search: Kuaicha 365 LawSkills API.
- Enterprise Info: Tianyancha AI.
- Optional: Qichacha, PKULaw, National Law Database.

## Boundary

Provider metadata may be registered before live integrations exist. Live provider calls stay disabled unless the target version explicitly enables them behind controlled gates. Dry-run remains enabled by default. Secret handling may report `key_loaded=true/false` and `key_source`, but must not read, return, mask, slice, log, or expose key values.

## Security Audit Cadence

For v7.25-v7.33, run basic checks at each sub-stage and defer full Codex Security audit to explicit final validation. The final audit must cover API key handling, secret boundary responses, raw content governance, provider live call gating, prompt boundaries, source trace, lawyer review, Skill training governance, open-case training separation, training artifact loading, Codex training run metadata, Skill Package validation metadata, internal training experience package metadata, practice load review metadata, controlled runtime loading metadata, output observation / feedback metadata, candidate pack metadata, next package draft metadata, lifecycle consolidation metadata, case-analysis output schema/workbench metadata, Skill final draft owner-only downloads, no Skill auto-publish, no open-case training, owner-only case/material workspace, fact preview / correction boundaries, legal analysis draft boundaries, owner-only output center aggregation, trial readiness issue log / quality reference boundaries, provider readiness live gate boundaries, OCR / Document provider live connection boundaries, unified live connection dashboard boundaries, Legal / Enterprise API live connection boundaries, public-link / email / external-delivery restrictions, final fact finding restrictions, final legal opinion restrictions, final report restrictions, frontend diagnostics, UI text safety, audit trail, and regression coverage.
