# Workspace System

Workspace System 是律师 AI 平台的案件执行系统。

负责：

* 案件管理
* 材料管理
* 事实抽取
* 法律分析
* 策略生成
* 文书生成
* 律师协作

Workspace System 不负责能力训练。

能力由 Training System 生产。

通过 Experience Package 向 Workspace System 提供能力。
