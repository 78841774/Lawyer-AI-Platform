# Training System

Training System 是律师 AI 平台的能力研发系统。

本系统负责：

- 律师经验提炼
- Skill 训练
- SkillOpt 优化
- Gate 验证
- Experience Package 构建
- Experience Package 发布

Training System 不直接处理线上案件。

Workspace System 不参与训练。

二者通过 Experience Package 连接。

- Architecture.md：训练系统总体架构
- Training-Pipeline.md：训练流水线
- SkillOpt-Workflow.md：Skill 优化流程
